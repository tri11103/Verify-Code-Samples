{% markdown %} ## Internationalization of Type Calcite Web is meant to be used by applications which are localized into hundreds of languages. Because of this, it is important that text displays properly in all languages. The main typeface used by Calcite Web is Avenir Next. Any languages that use the latin, cyrillic, or greek alphabets should render well with Avenir Next, but it doesn't contain characters required by several languages (including arabic, chinese, vietnamese, etc). Most of the languages not covered by Avenir Next don't pose a real problem. We can simply allow them to render in the browser default, and they are distinct enough that users won't feel as though something is wrong. The exception to this is Vietnamese, which uses standard Latin characters in addition to glyphs from the Latin Extended Additional set which are missing. This causes the missing characters to be rendered in a different font. This looks like a bug to the user. For this reason, we set the font family for pages in Vietnamese to Lucide Grande. Simply set the `lang` attribute of the html document to `vi` (the ISO-639 code for Vietnamese). You can preview how different languages render below with a sample set of characters from each language. Notice that vietnamese will be rendered in Lucide Grande as it is wrapped in an element with `lang` set to `vi`. {% endmarkdown %}

יידיש סרבול כתב של, סדר החול צילום בה. ב בהבנה חינוך הספרות שכל, ב עיצוב הגרפים היא. גם כלל הארץ יידיש מתמטיקה, חשמל ישראל לעריכת אנא או. ערכים קודמות חופשית אל עוד. מתן אם לערך שפות, דת מדינות קלאסיים מלא, את כתב ברית לציין פיסיקה. זאת הטבע לשון להפוך את, דת שכל היום משחקים ותשובות.

של מחליטה לעברית רב־לשוני צ'ט. את עזה ואמנות ויקימדיה בהיסטוריה, מה אחר צילום לראות אנציקלופדיה. אחד אל שנורו למאמרים, זאת ביולי ומדעים ומהימנה בה. שער תרבות והנדסה מבוקשים גם, פולנית משופרות ואלקטרוניקה מדע את. כיצד ביוני מדע על, מה כדי פנאי בלשנות. זאת של תוכל לעריכה, אם כדי משפטית לעריכת.
